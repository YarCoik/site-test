Локальная обработка данных

Весь код расширения выполняется исключительно в вашем браузере. YARBLOX не отправляет данные никуда. Никаких аналитических сервисов, никаких рекламных трекеров.

Разрешения браузера

YARBLOX запрашивает только минимально необходимые разрешения для работы на roblox.com.

Расширение использует localStorage только для сохранения ваших настроек (тема, параметры). Эти данные хранятся только на устройстве.
